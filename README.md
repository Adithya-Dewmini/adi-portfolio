# Nadun Portfolio

A premium creative developer portfolio built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Lucide Icons.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Customize

- Edit portfolio content in `src/lib/data.ts`
- Update sections in `src/components/sections.tsx`
- Change global styling in `src/app/globals.css`

## Deploy

Push to GitHub and import the repository into Vercel.
